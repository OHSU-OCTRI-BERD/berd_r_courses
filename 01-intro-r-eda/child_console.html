Variables

]

Missing values

Missing values are denoted as NA and are handled differently depending on the operation. There are special functions for NA (i.e. is.na(), na.omit()). .pull-left[

> x <- c(1, 2, NA, 5)
> is.na(x)
[1] FALSE FALSE  TRUE FALSE
> mean(x)
[1] NA
> mean(x, na.rm=TRUE)
[1] 2.666667

] .pull-right[

> x <- c("a", "a", NA, "b")
> table(x)
x
a b 
2 1 
> table(x, useNA = "always")
x
   a    b <NA> 
   2    1    1

]

Common console errors (2/2)

Object is not found

  • This happens when text is entered for a non-existent variable (object)

Example:

> hello
Error in eval(expr, envir, enclos): object 'hello' not found
  • Can be due to missing quotes
> install.packages(dplyr) # need install.packages("dplyr")
Error in install.packages(dplyr): object 'dplyr' not found
